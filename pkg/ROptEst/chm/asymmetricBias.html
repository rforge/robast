asymmetricBias(ROptEst)R Documentation

Generating function for asymmetricBiasType-class

Description

Generates an object of class "asymmetricBiasType".

Usage

asymmetricBias(name = "asymmetric Bias", nu = c(1,1) )

Arguments

name name of the bias type
nu weights for negative and positive bias, respectively

Value

Object of class "asymmetricBiasType"

Author(s)

Peter Ruckdeschel Peter.Ruckdeschel@uni-bayreuth.de

References

Rieder, H. (1994) Robust Asymptotic Statistics. New York: Springer.

Ruckdeschel, P. (2005) Optimally One-Sided Bounded Influence Curves. Mathematical Methods in Statistics 14(1), 105-131.

Kohl, M. (2005) Numerical Contributions to the Asymptotic Theory of Robustness. Bayreuth: Dissertation.

asymmetricBiasType-class
