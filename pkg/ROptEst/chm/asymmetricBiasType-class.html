asymmetric Bias Type

Description

Class of asymmetric bias types.

Objects from the Class

Objects can be created by calls of the form new("asymmetricBiasType", ...). More frequently they are created via the generating function asymmetricBias.

Slots

name:
Object of class "character".
nu:
Object of class "numeric"; to be in (0,1] x (0,1] with maximum 1; weights for negative and positive bias, respectively

Methods

nu
signature(object = "asymmetricBiasType"): accessor function for slot nu.
nu<-
signature(object = "asymmetricBiasType", value = "numeric"): replacement function for slot nu.

Extends

Class "BiasType", directly.
